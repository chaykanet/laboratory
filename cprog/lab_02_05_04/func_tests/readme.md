#  Входные данные:

```
Целые: размер массива, элементы массива.
```

#  Выходные данные:

```
Целые: кол-во уникальных чисел в массиве.
```

#  Позитивные тесты:

```
01 - размер массива 0 < n < N, целые элементы массива, все уникальные;
02 - размер массива n = N, целые элементы массива, все уникальные;
03 - размер массива 0 < n <= N, целые элменты массива, каждое число хотя бы 1 раз повторяется;
04 - размер массива 0 < n <= N, целые элменты массива, все элементы равны;
05 - размер массива 0 < n <= N, целые элменты массива, есть нулевые и отрицательные элементы;
06 - размер массива n = 1, один целый элемент;
```

#  Негативные тесты:

```
01 - размер массива n вне (0, N];
02 - размер массива n = 0;
03 - вместо размера n символ;
04 - размер массива 1 < n < N, целые элементы массива, один из которых символ.

```
