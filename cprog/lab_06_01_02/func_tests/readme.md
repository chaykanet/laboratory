#  Входные данные:

```
Текстовый файл, содержащий записи объектов, искомая начальная подстрока.
```
#  Выходные данные:

```
Вывод информации на экран по начальной подстроке.
```

#  Позитивные тесты:
```
01 - Подстрока отсутствует, неотсортированный список;
02 - Подстрока отсутствует, отсортированный список;
03 - Подстрока отсутствует, в списке имеются предметы с одной плотностью (устойчивость);
04 - Подстрока отсутствует, в списке один объект;
05 - Подстрока отсутствует, в списке два неотсортированных объекта;
06 - Подстрока "ALL", список не отсортирован;
07 - Подстрока "ALL", список отсортирован;
08 - Подстрока, в списке одно вхождение;
09 - Подстрока, в списке более одного вхождения;
10 - В списке максимальное кол-во объектов 15;
11 - В имени объекта 25 символов;
```

#  Негативные тесты:
```
01 - Файл не существует;
02 - Неверного кол-во аргументов при запуске;
03 - Пустой файл;
04 - Некорректный список (поля структуры неверно записаны);
05 - Нулевая масса объекта;
06 - Нулевой объем объекта;
07 - Отрицаетльные характеристики объекта;
08 - Подстрока, в списке нет вхождения;
09 - В списке больше максимального кол-ва объектов 16;
10 - В имени объекта больше 25 символов;
11 - Первая строка файла пустая;
```
