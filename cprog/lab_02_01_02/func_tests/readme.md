#  Входные данные:

```
Целые: размер массива. Элементы массива.
```

#  Выходные данные:

```
Вещественные: среднее арифметическое отрицательных элементов.
```

#  Позитивные тесты:

```
01 - размер массива 0 < n < N, целые элементы массива, несколько отрицательных элементов;
02 - размера массива 0 < n < N, целые элементы массива, все отрицательные элементы;
03 - размера массива максимальный n = N, целые элементы массива, несколько отрицательных элементов;
04 - размера массива 0 < n < N, целые элементы массива, один отрицательный элемент.
```

#  Негативные тесты:

```
01 - размер массива  n вне [0;N], целые элементы массива, есть отрицательные элементы;
02 - размера массива 0 < n < N, целые элементы массива, нет отрицательных элементов;
03 - вместо размера массива символ, целые элементы массива, есть отрицательные элементы;
04 - размера массива 0 < n < N, целые элементы массива, один из элементов символ; 
05 - нулевой массив n = 0;
```
