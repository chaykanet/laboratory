#  Входные данные:

```
Целые: Размеры матрицы. Элементы матрицы.
```

#  Выходные данные:

```
Целый: измененная матрица.
```

#  Позитивные тесты:

```
01 - размеры матрицы в допустимых значениях и меньше максимальных 10, матрица имеет более одного простого числа;
02 - размеры матрицы равны максимальным 10, матрица имеет более одного простого числа;
03 - размеры матрицы в допустимых значениях, матрица имеет одно простое число;
04 - размеры матрицы равны максимальным 10, все числа матрицы простые;
```

#  Негативные тесты:

```
01 - неверный ввод кол-ва столбцов матрицы;
02 - неверный ввод кол-ва строк матрицы;
03 - размер матрицы вне допустимых значений N вне (0, 10];
04 - размер матрицы вне допустимых значений M вне (0, 10];
05 - неверный ввод элементов матрицы;
06 - все элементы матрицы не простые числа;
07 - Элементы отрицательные, но по абсолютному значению простые.
```
