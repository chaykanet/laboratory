#  Входные данные:

```
Строка: Строка, содержащая дату формата [D]D MONTH YYYY.
```

#  Выходные данные:

```
Строка: Yes|No.
```

#  Позитивные тесты:

```
01 - введены корректные день, месяц и год;
02 - введены последний день февраля, год не високосный;
03 - введены последний день февраля, год високосный;
04 - введены последний день месяца, месяц, год;
05 - введены корректныe день, месяц с верхними регистрами символов, год;
06 - неверный формат даты;
07 - неверный день месяца (не целое);
08 - несуществующий день месяца;
09 - неверное название месяца;
10 - неверный год;
11 - нулевой год;
12 - Строка содержит лишние данные;
13 - Пустая строка;
14 - Отрицательный день месяца;
15 - Отрицательный год;
16 - Между днем месяцем и годом не пробелы;
17 - Между днем, месяцем и годом пробельные символы, но не пробелы;
18 - Между днем, месяцем и годом по несколько пробелов;
19 - Год меньше минимального 1000
20 - Год больше максимального 9999
```
