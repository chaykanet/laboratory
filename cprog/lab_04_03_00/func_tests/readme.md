#  Входные данные:

```
Строка: строка слов.
```

#  Выходные данные:

```
Строка: Строка, сформированная из слов в обратном порядке, в словах которой удалены все вхождения первых символов .
```

#  Позитивные тесты:

```
01 - в словах не более одного вхождения первого символа, слова различны;
02 - в словах более одного вхождения первого символа, слова различны;
03 - в словах все вхождения первого символа, слова различны;
04 - в словах не более одного вхождения первого символа, последнее слово повторяется;
05 - в словах более одного вхождения первого символа, последнее слово повторяется;
06 - в словах все вхождения первого символа, последнее слово повторяется;
07 - в строке два различных слова; 
08 - максимальный размер строки 256;
```

#  Негативные тесты:

```
01 - в строке слова, превышающие максимальный размер слова 16;
02 - В строке все одинаковые слова;
03 - один символ в строке;
04 - Строка превышающая максимальную длину 256
05 - В строке только пробельные символы.
06 - Пустая строка;
```
