#  Входные данные:

```
Целые: размер массива, элементы массива.
```

#  Выходные данные:

```
Целые: элементы нового массива.
```

#  Позитивные тесты:

```
01 - размер массива 0 < n < N, целые элементы массива, есть хотя бы 1 подходящий элемент;
02 - размер массива 0 < n < N, целые элементы массива, подходящий элементы идут подряд;
03 - размер массива максимальный N = 10, целые элементы массива, есть хотя бы 1 подходящий элемент;
04 - размер массива 0 < n < N, целые элементы массива, есть хотя бы 1 подходящий отрицательный элемент;
05 - размер массива 0 < n < N, все элементы массива подходящие;
```

#  Негативные тесты:

```
01 - размер массива  n вне (0, N];
02 - размер массива 0 < n < N, целые элементы массива, нет подходящих элементов;
03 - вместо размера массива символ;
04 - размер массива 0 < n < N, целые элементы массива, один из элементов символ;

```
