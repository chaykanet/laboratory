#  Входные данные:

```
Текстовый файл, массив целочисленных элементов.
```
#  Выходные данные:

```
Текстовый файл содержащий отфильрованный и отсортированный массив целых чисел.
```

#  Позитивные тесты:
```
01 - Ключ "f" отсутствует, 1 элемент;
02 - Ключ "f" отсутствует, 2 неотсортированных элемента;
03 - Ключ "f" отсутствует, 2 отсортированных элемента;
04 - Ключ "f" отсутствует, наполовину отсортированный массив;
05 - Ключ "f" отсутствует, отсортированный массив;
06 - Ключ "f" отсутствует, неотсортированный массив;
07 - Ключ "f", 2 элемента, второй отрицательный;
08 - Ключ "f", 2 отрицательных элемента;
09 - Ключ "f", массив, один отрицательный элемент в середине;
10 - Ключ "f", массив, один отрицательный элемент в начале;
11 - Ключ "f", массив, один отрицательный элемент в конце;
12 - Ключ "f", массив, нет отрицательных элементов;
13 - Ключ "f", массив, все отрицательных элементы;
```

#  Негативные тесты:
```
01 - Файл не существует;
02 - Неверное кол-во аргументов при запуске;
03 - Неверный ключ "f";
04 - Пустой файл;
05 - В массиве символы;
06 - В массиве нецелые числа;
07 - Ключ "f", 1 отрицательный элемент;
```
