#  Входные данные:

```
Целые: размер массива, элементы массива.
```

#  Выходные данные:

```
Целые: минимальное произведение двух соседних элементов.
```

#  Позитивные тесты:

```
01 - размер массива 0 < n < N, целые элементы массива;
02 - размер массива n = N, целые элементы массива;
03 - размер массива 0 < n <= N, целые элменты массива, есть отрицательные элементы;
04 - размер массива 0 < n <= N, целые элменты массива, есть нулевые элементы;
04 - размер массива 0 < n <= N, целые элменты массива, есть нулевые и отрицательные элементы;
```

#  Негативные тесты:

```
01 - размер массива n вне (0, N];
02 - размер массива n = 0;
03 - вместо размера n символ;
04 - размер массива 1 < n < N, целые элементы массива, один из которых символ.
05 - размер массива n = 1, целый элемент массива;

```
