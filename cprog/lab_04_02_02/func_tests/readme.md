#  Входные данные:

```
Строка: строка слов. 
```

#  Выходные данные:

```
Строки: пара <слово> yes|no.
```

#  Позитивные тесты:

```
01 - Строки не превышающие максимальный размер строки, слова не повторяются, все пары <слово> yes;
02 - Строки не превышающие максимальный размер строки, слова не повторяются, все пары <слово> no;
03 - Строки не превышающие максимальный размер строки, слова не повторяются;
04 - Строки не превышающие максимальный размер строки, слова повторяются;
05 - Строки размером 256, слова повторяются;
06 - Строки с одним словом, не превышающим максимальный размер слова;
07 - Строки не превышающие максимальный размер строки, слова разделены не только пробелами;
```

#  Негативные тесты:

```
01 - С строках слова, превышающие максимальный размер 16;
02 - Пустые строки;
03 - Строка превышающая максимальную длину 256
```
