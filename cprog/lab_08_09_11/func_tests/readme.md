#  Входные данные:

```
Текстовый файл, матрица вещественных чисел.
```
#  Выходные данные:

```
Текстовый файл, вещественные числа.
```

#  Позитивные тесты:
``` 
01 - ключ o, размер матрицы 1х2, положительные вещественные числа, однородное СЛАУ;
02 - ключ o, размер матрицы 1х2, положительные вещественные числа, неоднородное СЛАУ;
03 - ключ o, размер матрицы 2х3, положительные вещественные числа, однородное СЛАУ;
04 - ключ o, размер матрицы 2х3, положительные вещественные числа, неоднородное СЛАУ;
05 - ключ o, размер матрицы 3х4, отрицательные вещественные числа, неоднородное СЛАУ;
06 - ключ o, размер матрицы 4х5, числа с 6 нулями после запятой, неоднородное СЛАУ;
07 - ключ o, размер матрицы 4х5, нулевые элементы в матрице, неоднородное СЛАУ;
08 - ключ a, размер матриц 1x1, вещественные числа;
09 - ключ a, размер матриц 2х3, вещественные числа;
10 - ключ a, размер матриц 4х4, вещественные числа;
11 - ключ a, размер матриц 4х4, отрицательные вещественные числа;
12 - ключ m, размер матриц 2х3 и 3x2, вещественные числа;
13 - ключ m, размер матриц 4х4 и 4x4, вещественные числа;
14 - ключ m, размер матриц 4х4 и 4x4, отрицательные вещественные числа;
```

#  Негативные тесты:
```
01 - файл не существует;
02 - неверное кол-во аргументов при запуске;
03 - ключ o, пустой файл;
04 - ключ o, у матрицы символы вместо размеров;
05 - ключ o, у матрицы отрицательные размеры;
06 - ключ o, матрица не размеров nxn+1;
07 - ключ o, в матрице символы;
08 - в матрице кол-во элементов не соответствует nxm;
09 - ключ m, размеры матриц не удовлетворяют условию умножения;
10 - ключ a, размеры матриц не удовлетворяют условию сложения;
11 - ключ a, размеры матрицы не совпадают;
12 - ключ m, кол-во столбцов первой матрицы не соответствует кол-ву строк второй;
13 - ключ o, в матрице нулевой столбец;
14 - ключ o, в матрице нулевая строка;
15 - ключ o, в матрице строки повторяются;
16 - ключ o, в матрице строки отличаются на один множитель;
17 - ключ m, у матрицы отрицательные размеры;
18 - ключ m, в матрице символы;
19 - ключ m, файл пустой;
```
